# Sloviko

**Sloviko** is a lightweight vocabulary app for English learners. Open it — and you're straight into a quiz: a word in your native language or English, six translation options, tap. No sign-up, no onboarding, no menus — just endless tests with a smart repetition system.

The algorithm tracks every word in two memory registers: short-term (quick progress within a session) and long-term (built up over multiple encounters spaced across days). Learned words leave the rotation and return on growing intervals (7 → 14 → 30 → 90 days) — mirroring how real forgetting works.

Supported UI and translation languages: Ukrainian, Russian. The architecture makes adding new languages a single dictionary edit.

## Tech stack

- **Flutter** (iOS + Android), Dart 3.x
- **Riverpod** + `ChangeNotifier` for state
- **GoRouter** for navigation
- **sqflite** for local persistence (offline-first)
- **flutter_screenutil** for responsive sizing
- **google_fonts** (Inter)
- No code generation, hand-written models

## Architecture

Feature-first, three-layer:

```
lib/
├── core/         theme, router, locale, DI, icons
├── domain/       models, use-cases (no Flutter imports)
├── data/         sqflite DAOs, repositories, remote sources
└── view/         pages, providers, widgets — one folder per feature
```

The view layer never touches repositories or DAOs directly — only through use-cases.

## Run

```bash
flutter pub get
flutter run
```

```bash
dart run flutter_launcher_icons
```
